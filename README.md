Uma aplicação web **completa, moderna e responsiva** para gerenciar as finanças da família. Acompanhe receitas, despesas, metas de economia e visualize relatórios mensais — tudo diretamente no navegador, sem necessidade de servidor ou banco de dados.

## ✨ Funcionalidades

- ✅ Adicionar, editar e excluir receitas e despesas
- 🔄 Cálculo automático de valores normalizados (semanal, quinzenal, mensal, anual ou único)
- 🏷️ Filtro de despesas por 8 categorias pré-definidas
- 📊 Gráfico interativo de rosca com distribuição por categoria
- 📈 Indicadores em tempo real: total de receitas, despesas, saldo e % de economia
- 📅 Navegação entre meses e anos com histórico completo
- 💾 Backup e restauração via exportação/importação de arquivo `.json`
- 📱 Interface 100% responsiva (mobile, tablet e desktop)
- 🌙 Tema escuro moderno com animações suaves
- 💡 Dados salvos automaticamente no `localStorage` do navegador

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e acessível
- **CSS3** — Variáveis CSS, Flexbox, Grid, Design Responsivo e Transições
- **JavaScript (ES6+)** — Lógica de negócio, manipulação do DOM e persistência
- **Canvas API** — Renderização otimizada do gráfico de despesas
- **LocalStorage API** — Armazenamento local persistente sem backend
